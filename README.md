# Unity-Advanced-Weapon-System
A modular and feature-rich weapon system for Unity, including Raycast shooting, ammo management, and integrated audio/UI feedback.
A professional-grade weapon system framework designed for FPS/TPS projects in Unity.

## Key Features
- Hitscan (Raycast) Logic: Precision shooting mechanics for competitive gameplay.
- Physical Projectiles: Instantiate bullet prefabs for visual impact and ballistic feel.
- Ammo & Reload System: Full magazine management using Coroutines for non-blocking timers.
- Feedback Integrated: Built-in hooks for Recoil, Sound Effects, and Impact Particles.
- UI-Ready: Easy integration with TextMeshPro for real-time ammo tracking.

## Technical Notes
This system uses **Unity's New Input System** and is optimized for modularity, allowing it to be easily swapped between different weapon types.
